
Lista de Requerimiento
	Requerimientos funcionales

•	Ingresar a selección de actividad
	o	Muestra las posibles actividades a realizar
•	Visor de mapas
	o	Muestra los mapas que existen en el juego para facilitar su entendimiento y pensar estrategias.
•	Registro de las partidas
	o	Almacena un registro de las partidas y de esta forma seguir el progreso del jugador.
•	Misiones diarias
	o	Listado de misiones para practicar antes de entrar a jugar partidas normales.

	Requerimientos no funcionales

•	Rendimiento
	o	La aplicación funciona de manera fluida.
•	Mapas
	o	La aplicación almacena  los mapas.
•	Usabilidad
	o	La aplicación es predecible y sencilla de usar.
	
Sobre la aplicacion

Esta aplicación consta de 2 secciones.
 
•	Acceder a la información de la aplicación.
•	Acceder al menú principal.
En este menú tenemos 3 opciones:

1.	Permite ver los mapas del juego y familiarizarnos con ellos.
2.	Se podrá agregar el registro de las partidas que se jueguen y ver el historial de estas.
3.	Permite realizar prácticas sencillas para calentar antes de jugar partidas normales. En esta sección se colocan ciertas metas que el usuario debe cumplir.
